Leg Test Stand v1
=======================
<img src="images/leg_test_stand_1.png" width="600"> <br>  


Description
------------

2 x Actuator module  
1 x Lower leg with foot contact switch  
TI evaluation board electronics  
Vertical Range: 85cm
Hip height: 7cm - 92cm  
Weight of sliding carriage: 650g

-------------------
